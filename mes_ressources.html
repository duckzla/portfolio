Docker

Ce document explique comment créer, lancer et gérer des conteneurs avec Docker Compose, ainsi que la structure d’un fichier docker-compose.yml. Il détaille aussi la mise en réseau entre conteneurs (notamment application + base de données) et l’utilisation des variables d’environnement pour configurer les connexions. Enfin, il montre des exemples concrets avec PostgreSQL et la configuration des networks.

PhpMyAdmin

Ce document explique comment créer et gérer une nouvelle base de données et attribuer des droits utilisateurs via PhpMyAdmin, notamment pour plusieurs instances WordPress. Il aborde aussi la configuration réseau Docker nécessaire pour connecter correctement les services (ajout de networks). Enfin, il met en garde sur la gestion des permissions utilisateurs en cas d’erreur.

Protocole SSH

Guide pratique pour se connecter à distance à une machine via SSH. Explique les prérequis, les étapes de connexion (commande, acceptation de la clé, authentification), et comment se connecter à un container Docker via un port spécifique.

Clé ssh / github et container

Tutoriel complet pour configurer une clé SSH et l'utiliser avec GitHub et Docker. Couvre la génération de la paire de clés, l'ajout sur GitHub, la configuration Git locale, la création d'un fichier config SSH, et la mise en place d'un container Apache avec déploiement automatique depuis GitHub via Docker Compose.

Wordpress

Introduction à WordPress : ce que c'est, la différence entre WordPress.com (hébergé, simple) et WordPress.org (auto-hébergé, flexible), ce qu'on peut faire en frontend et en backend, ainsi que les avantages et inconvénients de la plateforme.

Installation d'une VM et FreeIPA

Ce document guide pas à pas la création d’une machine virtuelle sous VirtualBox, l’installation de Rocky Linux en mode serveur, puis la configuration réseau (IP fixe, hostname, SSH). Il détaille ensuite l’installation et la configuration complète de FreeIPA, incluant le firewall, DNS et Kerberos. L’objectif est de mettre en place un serveur d’authentification centralisé fonctionnel.
