Projet N°1 : Mise à disposition de services

Livrable scolaire documentant le déploiement d'une infrastructure de services via Docker. Le document présente l'organisation du projet (géré avec ClickUp), les outils utilisés (Notion, VS Code, Docker Desktop), puis détaille chaque application déployée en conteneur : deux bases de données (PostgreSQL et MySQL), des interfaces de gestion (Adminer, phpMyAdmin, Portainer), des outils métier (PIA pour la conformité RGPD, Baserow pour les données no-code, Projeqtor pour la gestion de projets, deux instances WordPress, Appflowy et N8n pour l'automatisation).

Chaque service est documenté avec son Docker Compose, ses ports, ses connexions et son rôle. Le document se conclut par une cartographie réseau illustrant l'architecture globale et les flux de communication entre les conteneurs.
